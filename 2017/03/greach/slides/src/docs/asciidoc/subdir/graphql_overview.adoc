=== What is GraphQL

[%step]
- Is a **query language**
- Is also a **server-side runtime** spec
- It uses a **type system** to define those queries

=== What is **NOT** GraphQL

[%step]
- Is not a web framework
- Is not about HTTP
- And definitely, definitely, definitely...
- **GraphQL is not REST**
